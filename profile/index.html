Learning Objectives

Professional Presentation Skills

Develop advanced techniques for creating compelling presentations using modern tools and design principles that engage and inform diverse audiences.

Critical Reasoning

Master logical thinking, argument analysis, and evidence evaluation to construct well-reasoned arguments and identify fallacies in reasoning.

Public Speaking Excellence

Theoretical Foundations

Performance Assessment

Regular presentation opportunities with peer feedback, self-reflection, and instructor evaluation to track progress and improvement.

IS 102: Basic Machine Learning

Discover the fundamentals of artificial intelligence and machine learning to solve real-world problems with data-driven solutions

Learning Objectives

Machine Learning Fundamentals

Understand core ML concepts including supervised, unsupervised, and reinforcement learning algorithms, and their practical applications in various domains.

Data Processing & Analysis

Master techniques for data collection, cleaning, preprocessing, and feature engineering to prepare datasets for machine learning models.

Algorithm Implementation

Learn to implement and optimize popular ML algorithms including linear regression, decision trees, neural networks, and clustering techniques.

Model Evaluation

Develop skills in assessing model performance using metrics, cross-validation, and techniques to prevent overfitting and underfitting.

Course Components

Programming Foundations

Build proficiency in Python programming, utilizing libraries like NumPy, Pandas, Scikit-learn, and TensorFlow for machine learning development.

Hands-on Projects

Apply theoretical knowledge through practical projects involving real datasets, from simple predictions to complex pattern recognition tasks.

Industry Applications

Explore ML applications across industries including healthcare, finance, e-commerce, and technology through case studies and examples.

Ethics & Best Practices

Understand responsible AI development, including bias detection, fairness considerations, and ethical implications of machine learning systems.
